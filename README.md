# PokoBuild3D

## Live Site

[PokoBuild3D Site](https://chimeragaming.github.io/PokoBuild3D/)

## What This Site Is

PokoBuild3D is a place for Pokopia players to share builds, pictures, real life creations, and visual guides in one spot. It is made to be simple to browse, simple to search, and easy to post in.

## What You Can Post

- `3D Model` for browser made builds or uploaded 3D model files
- `Picture` for screenshots, reference images, and image galleries
- `Real 3D` for handmade builds, painted pieces, and 3D printed creations
- `Tips & Tricks` for guide images, labeled info sheets, and visual how to posts

## What The Site Does

- lets people browse community builds
- lets people search by tags and location
- lets people save favorites
- lets people chat in the live chat dock
- lets people build inside the browser editor
- lets people upload image based posts
- lets people track Pokopia block coverage in the game block tracker

## How Posting Works

1. Pick the kind of post you want to make.
2. Add the title, description, and location.
3. Choose `My Creation` or `Online Creation`.
4. If it is from online and you know the owner, add the creator name so credit shows correctly.
5. Upload your files, or continue into the editor if you are making a browser 3D build.
6. Publish the post.

## Main Areas Of The Site

- `Home` for featured posts and quick entry points
- `Catalog` for browsing and search
- `Game Blocks` for checking which Pokopia blocks are already covered in the demo set
- `Create` for making a new post
- `Editor` for browser based 3D building
- `Favorites` for saved posts
- `Profile` for your account and your uploads

## Post Types

### 3D Model

Use this if the build should be viewable in 3D.

- You can continue into the browser editor
- Or upload a GLB or glTF file
- Good for full builds, layouts, and viewer ready creations

### Picture

Use this if the post is mostly screenshots or reference images.

- Good for galleries
- Good for simple build references
- Good for posts that do not need a 3D viewer

### Real 3D

Use this for off screen creations.

- handmade pieces
- clay work
- papercraft
- painted models
- 3D printed creations

This post type also lets people add shop links.

### Tips & Tricks

Use this for image based help posts.

- labeled guide images
- visual breakdowns
- step by step image sheets
- quick tips saved as images
